## Description

This package is a collection of useful TypeScript decorators for Klasa projects. Notable `@ApplyOptions` and `@CreateResolvers` will let you bypass the need for a class constructor in many cases.

## API Documentation

For the full API documentation please refer to the TypeDoc generated [documentation](https://skyra-project.github.io/decorators/).
